## v0.0.8

> [!info] What's New in v0.0.8
> 
> - Reorganized the **Mathematics** directory to improve navigation and consistency.
> - Refined the [[Mathematics|Mathematics]] Map of Content (MoC).
> - Added dedicated Maps of Content for [[Linear Algebra]], [[Calculus]], and [[Differential Equations]].
> - Added GATE PYQs (2010–2026) in Markdown format. See `Resources/Question Paper/GATE/MD/`
> - Added `onlyQ` folder in `Resources` folder. [[onlyQ_ee_2026]] contains only questions no tags no topic links.
> - Added images corresponding to the question papers under `Resources/Images/Question Paper`.
> - Added the `embeds.css` CSS snippet in `.obsidian/snippets` to enable seamless embedding with `![[...]]`. To enable it, navigate to **Settings → Appearance → CSS snippets** and toggle the snippet on.
> - Started recording the `updated` property to track the date a note was last modified.

### Improved

- Enhanced the [[Numerical Methods]] Map of Content.
- Expanded cross-links between related topics to improve discoverability.

### Fixed

- Resolved broken frontmatter links.
- Removed Mermaid mind maps that were not rendering correctly.
