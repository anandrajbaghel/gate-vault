### Decibels (dB)
*Signal Strength & Gain Measurement*

A logarithmic unit used to express ratios such as power gain or attenuation, given by $10 \log_{10} \left(\frac{P_2}{P_1}\right)$ for power and $20 \log_{10} \left(\frac{V_2}{V_1}\right)$ for voltage.

---

|         | Formula                                       |     |      | Formula                                              |
| ------- | --------------------------------------------- | --- | ---- | ---------------------------------------------------- |
| Power   | $$10 \log_{10} \left(\frac{P_2}{P_1}\right)$$ |     | Gain | $$10 \log_{10} \left(\frac{P_{out}}{P_{in}}\right)$$ |
| Voltage | $$20 \log_{10} \left(\frac{V_2}{V_1}\right)$$ |     | Loss | $$10\log_{10} \left( \frac{P_{in}}{P_{out}}\right)$$ |
