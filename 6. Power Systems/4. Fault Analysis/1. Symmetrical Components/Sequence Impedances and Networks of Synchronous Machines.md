### Sequence Impedances and Networks of Synchronous Machines
#power-systems/fault-analysis #synchronous-machines #sequence-networks

> A synchronous machine, being a rotating device with non-uniform rotor construction, presents a different impedance to each of the three sequence currents. To perform unsymmetrical fault analysis using [[Concept of Symmetrical Components|symmetrical components]], the machine must be represented by three distinct **sequence networks**, one for each sequence component.

#### 1. Positive Sequence Network
#positive-sequence-network

This network represents the machine's behavior under normal, balanced, three-phase conditions.

*   **Impedance ($Z_1$):** The positive sequence impedance is the same as the standard impedance of the machine. For fault studies, we are interested in the initial fault current, so we use the **subtransient reactance**, $X_d''$. Armature resistance is usually neglected.
    $$ Z_1 = R_a + jX_1 \approx jX_d'' $$
*   **Network Model:** The positive sequence network is the only one that contains a voltage source. This source is the machine's internal induced EMF ($E_a$), which is a balanced, positive sequence set of voltages generated by the rotor's main field.
*   **Equation:** The terminal voltage for phase 'a' is given by:
    $$\boxed{\quad V_{a1} = E_a - I_{a1}Z_1 \quad}$$
    where $E_a$ is the pre-fault internal voltage, typically assumed to be $1.0 \angle 0^\circ$ p.u. if the machine is unloaded.

---
#### 2. Negative Sequence Network
#negative-sequence-network

This network represents the machine's response to unbalanced conditions that create negative sequence currents.

*   **Impedance ($Z_2$):** Negative sequence currents create a magnetic field that rotates at synchronous speed but in the **opposite direction** to the rotor. This reverse-rotating field induces currents in the rotor circuits (damper and field windings) at **twice the system frequency** (e.g., 100 Hz for a 50 Hz system). The impedance offered to these currents is the negative sequence impedance.
    *   The reactance, $X_2$, is approximately the average of the direct and quadrature axis subtransient reactances:
        $$Z_2 = R_a + jX_2 \approx j\left(\frac{X_d'' + X_q''}{2}\right)$$
*   **Network Model:** The main rotor field rotates with the rotor and cannot induce negative sequence voltages in the stator. Therefore, the negative sequence network **does not contain any voltage source**. It is a passive network.
*   **Equation:** The terminal voltage is simply the voltage drop across the impedance:
    $$\boxed{\quad V_{a2} = -I_{a2}Z_2 \quad}$$

---
#### 3. Zero Sequence Network
#zero-sequence-network

This network represents the machine's response to faults that involve ground, allowing zero sequence currents to flow.

*   **Impedance ($Z_0$):** The three zero sequence currents are in phase. They produce a pulsating magnetic field instead of a rotating one. The flux path for this field is very different from that of the positive or negative sequence fields, resulting in a very different reactance value, $X_0$. $X_0$ is typically much smaller than $X_1$ or $X_2$.
    $$ Z_0 = R_a + jX_0 $$
* **Network Model:** Zero sequence currents can only flow if there is a path through the machine's neutral connection. The main field cannot produce zero sequence voltages. Therefore, the zero sequence network is **passive and has no voltage source**.
* **Effect of Neutral Grounding:** The impedance of the neutral connection, $Z_n$, is critical. Since the total current flowing through the neutral is $I_n = I_a + I_b + I_c = 3I_{a0}$, its effect in the phase 'a' equivalent network is modeled as an impedance of $3Z_n$.
    * If the neutral is **solidly grounded**, $Z_n=0$.
    * If the neutral is **ungrounded**, $Z_n=\infty$, and the zero sequence network is an open circuit ($I_{a0}=0$).
* **Equation:** The terminal voltage is:
    $$\boxed{\quad V_{a0} = -I_{a0}Z_0 \quad}$$

---
#### Summary of Sequence Networks

| Network Type        | Voltage Source | Impedance                                        | Notes                                                        |
| ------------------- | :------------: | ------------------------------------------------ | ------------------------------------------------------------ |
| **Positive ($Z_1$)**  |   **Yes** ($E_a$)   | $Z_1 \approx jX_d''$                             | Represents the balanced, pre-fault state of the machine.     |
| **Negative ($Z_2$)**  |       No       | $Z_2 \approx j(X_d''+X_q'')/2$                    | Represents response to unbalanced currents (reverse field).  |
| **Zero ($Z_0$)**        |       No       | $Z_0 \approx jX_0$ (depends on winding)           | Represents response to ground faults. Depends on $Z_n$.      |
