### Zero-Input Response (ZIR)
#signals-and-systems/time-response #control-system/state-space

> The ==**Zero-Input Response (ZIR)** of a system is the response (output) caused *entirely* by the system's initial conditions== (the energy already stored in its inductors, capacitors, or mechanical equivalent elements) when the external input is strictly zero. ==It is often referred to as the **Free Response** or **Unforced Response**.==

#### 1. The Total Response Decomposition
#signals-and-systems/lti

For any [[LTI|Linear Time-Invariant (LTI) system]], the [[Linearity#The Principle of Superposition|Principle of Superposition]] dictates that the total response can be cleanly separated into two independent components:
1.  **Zero-Input Response (ZIR):** Response due to initial conditions alone (Input $u(t) = 0$).
2.  **[[Zero-State Response (ZSR)]]:** Response due to the input alone (Initial States $x(0) = 0$).

$$\boxed{\quad \text{Total Response } y(t) = y_{ZIR}(t) + y_{ZSR}(t) \quad}$$

---
#### 2. ZIR in Differential Equations and Laplace Domain
#control-system/laplace

In classical differential equation analysis, setting the input to zero leaves the **homogeneous differential equation**.

When solving using the Laplace Transform, the ZIR emerges directly from the initial condition terms generated by the derivative property:
$$\mathcal{L}\left\{\frac{dy}{dt}\right\} = sY(s) - y(0^-)$$
$$\mathcal{L}\left\{\frac{d^2y}{dt^2}\right\} = s^2Y(s) - sy(0^-) - y'(0^-)$$

If you collect all terms in the Laplace domain and set the input $U(s) = 0$, you get:
$$\boxed{\quad Y_{ZIR}(s) = \frac{\text{Terms containing } y(0^-), y'(0^-), \dots}{\text{Characteristic Polynomial}} \quad}$$

> [!memory] Key Insight
> The denominator of $Y_{ZIR}(s)$ is the characteristic polynomial of the system. Therefore, the shape of the Zero-Input Response (e.g., exponentially decaying, oscillating) is entirely determined by the **poles of the system**.

> [!examtip] Decoupling Poles from Initial Conditions
> A common point of confusion is assuming that because a Transfer Function $H(s)$ requires zero initial conditions ($y(0^-) = 0, y'(0^-) = 0$), the roots of its denominator change if the initial conditions are non-zero. 
> 
> They do **not**. 
> 
> * **The Core Mechanism:** When solving an LCCDE with Laplace transforms, initial condition terms exclusively populate the *numerator* of the Zero-Input Response ($Y_{ZIR}(s)$). The denominator remains the exact same characteristic polynomial ($D(s) = 0$) regardless of the system's initial energy state.
> 
> > [!memory] The Takeaway
> > Initial conditions only affect the scaling coefficients of the time-domain modes, never the locations of the poles themselves.

#### 3. ZIR in State-Space Analysis
#state-space/solution

In the state-space representation, setting the input vector $u(t) = 0$ leaves the homogeneous state equation:
$$\dot{x}(t) = Ax(t)$$

The complete solution for the state vector $x(t)$ subject to an initial condition $x(0)$ is derived using the [[State Transition Matrix (STM)]], $\Phi(t) = e^{At}$.

The **Zero-Input Response of the state vector** is simply:
$$\boxed{\quad x_{ZIR}(t) = \Phi(t) x(0) = e^{At} x(0) \quad}$$

If we need the actual output $y(t) = Cx(t) + Du(t)$, and knowing $u(t)=0$:
$$\boxed{\quad y_{ZIR}(t) = C \Phi(t) x(0) = C e^{At} x(0) \quad}$$

---
#### 4. ZIR and Internal Stability
#stability/definitions

The Zero-Input Response is the definitive measure of a system's **Internal Stability** (or Asymptotic Stability).
*   A linear system is **Asymptotically Stable** if and only if its Zero-Input Response decays to zero as time goes to infinity, for *any* finite initial condition.
    $$\lim_{t \to \infty} y_{ZIR}(t) = 0$$
*   This occurs mathematically when all eigenvalues of the matrix $A$ (or all poles of the transfer function) have strictly negative real parts.

---
#### 5. Distinction: ZIR vs. Natural Response
#signals-and-systems/definitions

A common semantic trap in GATE involves confusing ZIR with "Natural Response".
*   **Zero-Input Response (ZIR)** is strictly defined by the physical state: Input is zero, response is due *only* to initial conditions.
*   **Natural Response** (in classical texts) refers to all terms in the total response that share the form of the characteristic modes (e.g., $e^{p_1 t}, e^{p_2 t}$). The application of an input (ZSR) can *also* excite these natural modes. 
*   Therefore, the **Natural Response** often contains the entire ZIR *plus* a portion of the ZSR.
